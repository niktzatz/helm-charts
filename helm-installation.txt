==========================================================
== Helm installtion on Windows
== based on https://helm.sh/docs/intro/quickstart/
==========================================================
1. Get the latest stable release from (download the binary format):
	
	https://github.com/helm/helm/releases
	
2. 	Unzip under E:\_Installed Programs_\Helm\windows-amd64

3. Add to Windows' PATH

4. Add the following environment variables (run "helm" in terminal for more details):

	HELM_CONFIG_HOME=E:\_Workspaces_\Helm\configuration
	HELM_DATA_HOME=E:\_Workspaces_\Helm\data

5. Open a new terminal and run:

	helm
		
	
6. Add a chart repository	

	(Check Artifact Hub for available Helm chart repositories. https://artifacthub.io/packages/search?kind=0)
	
	helm repo add bitnami https://charts.bitnami.com/bitnami
	
	helm search repo bitnami


7. Install an Example Chart
	
	helm repo update 
	
	helm install bitnami/mysql --generate-name

8. Uninstall a Release
	
	helm list 
	
	find the release and uninstall it:
	
	helm uninstall mysql-1612624192
